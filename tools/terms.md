    + 仓库：用于存放文件，以.git为结尾。克隆到本地则为一个文件夹。
    + remote：服务器端仓库的别名。默认为origin。
    + clone：将服务器端的库克隆到本地。
    + git add：将更改的文件加入缓存区
    + git commit：将缓存区的更改在本地提交。
    + push：将本地库提交的更改推送到远程库。
    + pull：将服务器端的更改合并到本地。
